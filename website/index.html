Introduction

Use the application generator command-line tool for a new android project to create an application skeleton simply and quickly.

Download and Run:

$ git clone git@github.com:kimkevin/AndroidStarterKit.git
$ cd AndroidStarterKit

# Add Path Permanently
$ export PATH=$PATH:$(pwd) >> ~/.bash_profile

For example, the following command generate the Android project of the following path which included in the GridView and RecyclerView for layouts and the Firebase Analytics and Crash Reporting. Use the following command to do so:

$ ask -l gv,rv -t fa,fc /path/to/android/project

If the command-line doesn’t have the project path for [dir] option, the new project will created in an output folder.

$ ask -l gv,rv -t fa,fc

Sync it with Gradle and run it.

Check the command options with the -h(--help) option:

Usage: ask [options] [dir]
Options:
  -l, --layout <view>...  add <view> support: sv(ScrollView), rv(RecyclerView), lv(ListView), gv(GridView)
                          defaults to ScrollView
  -i, --icon              tab icon instead of text more than 2 views

  -t, —-tool <lib>...     add <lib> support: fa(FirebaseAnalytics), fc(FirebaseCrashReporting)

  -h, --help              output usage information

Layout

The goal of the tool was to create UI deisgn simply and easier for developers. We’re looking forward to expanding this project in the future to help share their custom views which consist of only code without any access to image resources on a GUI supported website. Use the following command to do so.

$ ask -l <view>... /path/to/android/project

Available Layout commands

ask -l gv or GridView

ask -l rv or RecyclerView

ask -l lv or ListView

ask -l sc or ScrollView

If you want to use the multiple layouts

ask -l gv,rv (GridView and RecyclerView)

Generated the ViewPager which has a SlidingTabLayout in case of the multiple layouts.

And if you want to show the icon of tab instead of text

ask -l gv,rv --icon

Icon option generates SlidingIconTabLayout which has an icon instead of a text on the tab

Tool - Third Party Support

Tool option offers some useful tools to help manage your project fast and easily such as Firebase. Use the following command to do so.

$ ask -t <lib>... /path/to/android/project

Firebase

Firebase gives you the tools and infrastructure you need to build better apps and grow successful businesses.

Create new project to add Firebase to your Android application or import google project if you already have an existing Google project in Firebase console. You’ll download a google-service.json file and copy it into your project’s application module folder. For more details

Firebase Console :

Analytics
Crash Reporting

Available Firebase commands

ask -t fa or FirebaseAnalytics

Firebase Analytics is a free app measurement solution that provides insight on app usage and user engagement.

ask -t fc or FirebaseCrashreporting

Crash Reporting creates detailed reports of the errors in your app.

If you want to use the multiple tools

ask -t fa,fc (Analytics, Crash Report)

Fabric

Fabric is a platform that helps your mobile team build better apps, understand your users, and grow your business.

Create new project to add Fabric console to your Android application. If you are using the Android Studio or IntelliJ for development environment, install Fabric IDE plugin. For more details

Fabric Console :

Answers
Crashlytics

Available Fabric commands

ask -t an or Answers

Answers is Fabric’s mobile analytics SDK. With one click, you can enable real-time analytics that help you understand what's happening in your app. Fabric provides insights into your core goals, such as growth, retention, and engagement

ask -t cr or Crashlytics

Fabric Crashlytics is the most powerful, yet lightest weight crash reporting solution.

If you want to use the multiple tools

ask -t an,cr (Answers, Crashlytics)

Stetho

Stetho is a sophisticated debug bridge for Android applications. When enabled, developers have access to the Chrome Developer Tools feature natively part of the Chrome desktop browser. Developers can also choose to enable the optional dumpapp tool which offers a powerful command-line interface to application internals.
Analytics
Crash Reporting

Features

Chrome DevTools provides the navigation to chrome://inspect, Network Inspection, Database Inspection, View Hierarchy, Dumpapp, and Javascript Console.

Available commands

ask -t st or Stetho
